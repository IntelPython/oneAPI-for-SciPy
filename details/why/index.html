Why should Scientific Python community embrace oneAPI

  1. oneAPI is an open, cross-industry, standards-based, unified, multiarchitecture, multi-vendor programming model.
  2. DPC++ compiler is being developed in open-source, see http://github.com/intel/llvm, and is being upstreamed into LLVM project itself.
  3. Open source compiler supports variety of backends, including Level-Zero, OpenCL, CUDA, and HIP.
  4. oneAPI Math Kernel Library (oneMKL) Interfaces supports a collection of third-party libraries associated with supported backends.
